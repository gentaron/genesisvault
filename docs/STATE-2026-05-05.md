## 3. GitHub Actions Workflow

File: `.github/workflows/daily-post.yml`

```yaml
name: Genesis Vault — Multi-Agent Daily Post

on:
  schedule:
    - cron: '30 11 * * *'   # 19:30 MYT (UTC+8) daily
  workflow_dispatch:

permissions:
  contents: write

jobs:
  create-post:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      - name: Run Multi-Agent Pipeline
        env:
          GEMINI_API_KEY: ${{ secrets.GEMINI_API_KEY }}
        run: node scripts/auto-post.mjs
      - name: Commit and push
        run: |
          git config --local user.email "action@github.com"
          git config --local user.name "Mina Eureka Ernst [Bot]"
          git add src/content/posts/*.md
          git diff --quiet && git diff --staged --quiet || (git commit -m "New post: $(date +'%Y-%m-%d') — Multi-Agent AI Pipeline" && git push)
```

**Secrets in use**: `GEMINI_API_KEY`

## 7. Scripts

### auto-post.mjs (Multi-Agent Pipeline)
- 5-agent pipeline: VE-005 Nova → VE-001 Lena → VE-003 Chloe → VE-002 Sophia → VE-006 Iris
- Uses `gemini-2.5-flash-lite` (15 RPM / 1000 RPD) and `gemini-2.5-flash` (10 RPM / 250 RPD)
- **Note**: `gemini-2.0-flash` was removed by commit `1447141`
- Theme balance analysis across 9 categories
- Exponential backoff (10s base, 3 retries max)
- XML/CDATA parser for legacy gensnotes content
- Fallback template system when API fails

### nostr-broadcast.mjs (Nostr NIP-23)
- Long-form Content (kind: 30023) broadcaster
- Relays: `wss://relay.damus.io`, `wss://nos.lol`, `wss://relay.snort.social`, `wss://relay.nostr.band`
- Uses `nostr-tools/pure` (SimplePool, finalizeEvent, verifyEvent)
- Required env: `NOSTR_PRIVATE_KEY` (64-char hex)
- 15-second timeout per relay

### ipfs-archive.mjs (Pinata IPFS)
- Pins markdown files to IPFS via Pinata Free Tier (1GB)
- CIDv1 output
- Required env: `PINATA_JWT`
- Gateways: `ipfs.io`, `gateway.pinata.cloud`
